如果有多个商户号需要操作的话将会是一个不小的麻烦，因此实现一个工具来解决这种问题还是很有必要的。把多个商户号汇集到一处管理，想操作哪个商户号就选哪个商户号即可。

可用于现金营销、分销返佣、行政补贴、行政奖励、保险理赔、佣金报酬、企业报销、企业补贴、服务款项、采购货款等商户向个人转账的场景。

现在来看看这个工具具体都有哪些功能。

#### 商户信息

这里录入的是商户号相关信息，可以管理多个商户号。

![Image text](https://pic2.zhimg.com/80/v2-55278c7b910fa607730b619a0d2aeb91_720w.webp)

要准备商户名称、商户平台apiV3密钥、商户号、商户appId、商户API证书序列号、微信支付平台证书序列号、商户API证书、微信支付平台证书，这些参数信息到微信支付商户后台获取。

![Image text](https://pic2.zhimg.com/80/v2-c57e9774d57c4e628fc18b083961ae09_720w.webp)

#### 转账入口

支持单笔转账和批量转账，当批量转账只有一条数据的时候也相当于单笔转账。

![Image text](https://pic4.zhimg.com/80/v2-530348a4ed91257de396ceff661696af_720w.webp)

单笔转账，商户信息选择要用哪个商户号信息、输入批次名称、批次备注、转账金额、转账备注、收款用户openId、收款方真实姓名。收款用户openId需要和商户appId对应，也就是说openId是由对应的appId获取到的。

![Image text](https://pic4.zhimg.com/80/v2-f378e00097567731f12b3138cd33f58f_720w.webp)

批量转账，商户信息选择要用哪个商户号信息、输入批次名称、批次备注。上传批次名单，批次名单需要下载模板按照模板要求填写转账信息，信息包括收款用户openId、收款方真实姓名、转账金额、转账备注。同理，收款用户openId需要和商户appId对应，也就是说openId是由对应的appId获取到的。

#### 下载电子回单

支持下载批次回单和明细回单。

下载批次回单，先点击申请批次回单，仅支持已完成状态且至少一笔成功记录的批次获取电子回单，申请批次回单成功后，再点击下载批次回单，即可下载到pdf格式的批次回单。

![Image text](https://pic3.zhimg.com/80/v2-57701e54b88464b7c5c44b08273c3f6a_720w.webp)

下载明细回单，先点击申请明细回单，仅支持转账成功状态的明细获取电子回单，申请明细回单成功后，再点击下载明细回单，即可下载到pdf格式的明细回单。

![Image text](https://pic4.zhimg.com/80/v2-c73b2622ee96d8aeb31fdc32cf63745b_720w.webp)

微信商户号后台可以设置转账额度、免密额度、请求IP地址来保障商户资金安全，为操作保驾护航。
